# Face-Recognition-Attendance


A Smart Attendance System that uses face recognition to automatically mark
attendance in an Excel (CSV) file.

## Features
- Face detection and recognition
- Automatic attendance marking
- One attendance per student per day
- Excel (CSV) based attendance record

## Technologies Used
- Python
- OpenCV
- LBPH Face Recognition

## How to Run
1. Clone the repository
2. Install required libraries:
   pip install -r requirements.txt
3. Add student images inside the dataset folder
4. Run the project:
   python attendance.py

## Note
For privacy reasons, real student images are not included.
